# Прогнозирование конечных свойств композиционных материалов

Выпускная квалификационная работа по курсу «Data Science Pro», МГТУ им. Н. Э. Баумана, 2025.

Репозиторий содержит Streamlit-приложение для прогнозирования механических свойств полимерных композиционных материалов (модуль упругости при растяжении и прочность при растяжении) и рекомендации оптимального соотношения матрица-наполнитель на основе 10 технологических параметров компонентов и параметров нашивки.

## Структура репозитория

```
composite-properties-predictor/
├── data/                      # исходные датасеты
│   ├── X_bp.xlsx
│   └── X_nup.xlsx
├── models/                    # обученные модели (создаются train_models.py)
│   ├── ridge_modulus.joblib
│   ├── rf_strength.joblib
│   ├── scaler_23.joblib
│   ├── mlp_ratio.pt
│   ├── scaler_mlp_X.joblib
│   ├── scaler_mlp_y.joblib
│   └── metadata.json
├── app.py                     # Streamlit-приложение
├── train_models.py            # обучение и сериализация моделей
├── requirements.txt
└── README.md
```

## Установка

Требуется Python 3.10 или выше.

```bash
git clone https://github.com/digger32/composite-properties-predictor.git
cd composite-properties-predictor

python -m venv venv
source venv/bin/activate         # Linux / macOS
# venv\Scripts\activate          # Windows

pip install -r requirements.txt
```

## Подготовка датасета

Исходные файлы `X_bp.xlsx` и `X_nup.xlsx` не включены в репозиторий и должны быть помещены в директорию `data/` вручную.

## Обучение моделей

```bash
python train_models.py
```

Скрипт считывает датасеты из `data/`, обучает три модели и сохраняет артефакты в `models/`:

| Модели | Назначение |
|---|---|
| `ridge_modulus.joblib` | Ridge-регрессия для прогноза модуля упругости при растяжении |
| `rf_strength.joblib` | Random Forest для прогноза прочности при растяжении |
| `scaler_23.joblib` | StandardScaler для входов моделей раздела 2.3 |
| `mlp_ratio.pt` | state_dict нейронной сети для рекомендации соотношения |
| `scaler_mlp_X.joblib` / `scaler_mlp_y.joblib` | StandardScaler для входов и выхода MLP |
| `metadata.json` | Имена признаков, метрики, версия |

## Запуск приложения

```bash
streamlit run app.py
```

Приложение открывается в браузере по адресу `http://localhost:8501` и содержит две независимые вкладки:

* **Прогноз механических свойств** — ввод 11 параметров и вывод двух целевых величин.
* **Рекомендация соотношения матрица-наполнитель** — ввод 10 параметров и вывод рекомендуемого соотношения.

## Ограничения моделей

По результатам 10-кратной кросс-валидации на имеющемся датасете (1023 объекта) ни одна из построенных моделей не превосходит базовый прогноз средним значением (DummyRegressor) на статистически значимую величину. Максимальный коэффициент корреляции Пирсона между входными признаками и целевыми переменными не превышает 0,075. Приложение носит демонстрационный характер и не предназначено для принятия производственных решений без дополнительной валидации на расширенном датасете с физико-химическими признаками. Подробное обоснование приведено в подразделах 2.3 и 2.4 пояснительной записки.
